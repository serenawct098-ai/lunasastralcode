## 【嚴格語言與輸出規範】（出現違規詞即重寫，無例外）

1. **基本語言**：所有內容（包括畫面腳本、卡字、正文、特效花字、字幕）強制使用繁體中文書面語，嚴禁廣東話口語（如：嘅、咁、唔、係咪、點解）。
2. **AI口吻嚴禁清單**（必須逐條檢查）：
    - **轉折/對比句型**——嚴禁：「不是……而是……」、「儘管……但……」、「無論是……還是……」
    - **萬用起手式與破折號濫用**——嚴禁：句末/句中濫用破折號（——）做補充說明、「先說結論」、「一句話總結」、「總而言之」、「值得注意的是」、「讓我們來看看」、「原因很簡單」
    - **絕對理性說教詞彙**——嚴禁：「本質上」、「其實」、「很清楚」、「很簡單」、「很現實」、「說白了」
3. **禁用療癒雞湯句**：不用「我懂你的感受」「你已經撐太久了」「我會在這裡接住你」等空話。情感靠具體場景與細節傳達。
4. **機械連接詞節制**：「此外」「因此」「同時」「然而」「首先」「最後」「總之」「換句話說」全篇合計不超過2次。優先用語意自然過渡。
5. **標點規範**：破折號（——）僅限語意突然轉折或強調停頓，全篇最多1次。不用破折號補充說明（改用逗號/括號）。
6. **語氣節奏**：語氣自然口語化，多用助語詞（呢／吧／喔）帶人情味（每段不超過3個）。句子長短交錯，每段控制在3-5句內。
