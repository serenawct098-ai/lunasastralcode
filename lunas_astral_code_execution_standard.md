## 【嚴格語言與輸出規範】（出現違規詞即重寫，無例外）

1. **基本語言**：所有內容（包括畫面腳本、卡字、正文、特效花字、字幕）強制使用繁體中文書面語，嚴禁廣東話口語（如：嘅、咁、唔、係咪、點解）。
2. **AI口吻嚴禁清單**（必須逐條檢查）：
    * **轉折/對比句型**——嚴禁使用：「不是……而是……」、「儘管……但……」、「無論是……還是……」
    * **萬用起手式與破折號濫用**——嚴禁使用：句末/句中濫用破折號（——）做補充說明、「先說結論」、「一句話總結」、「總而言之」、「值得注意的是」、「讓我們來看看」、「原因很簡單」
    * **絕對理性說教詞彙**——嚴禁使用：「本質上」、「其實」、「很清楚」、「很簡單」、「很現實」、「說白了」
3. **禁用療癒雞湯句**：不用「我懂你的感受」「你已經撐太久了」「我會在這裡接住你」等空話。情感靠具體場景與細節傳達。
4. **機械連接詞節制**：「此外」「因此」「同時」「然而」「首先」「最後」「總之」「換句話說」全篇合計不超過2次。優先用語意自然過渡。
5. **標點規範**：破折號（——）僅限語意突然轉折或強調停頓，全篇最多1次。不用破折號補充說明（改用逗號/括號）。
6. **語氣節奏**：語氣自然口語化，適量用助語詞（呢／吧 / 啊）帶人情味（每段不超過1個）。句子長短交錯，每段控制在3-5句內。

## Part 3：素材結構模板參考（必須有新創意，不可照搬舊題材）

### 1. 心理測驗（測驗互動型｜Reels）
* **【Hook】拋出誘餌**：影片開頭或封面用一句話點出大眾好奇的痛點。
* **【正文】動態互動**：畫面快速閃過吉凶字卡，文字說明測驗規則，強調「截圖即答案，第一直覺最準」。
* **【CTA】衝高留言與引導擴散**：
    * 留言＆讚好：「雙擊螢幕點個讚，並把截圖結果打在留言區，看看誰跟你抽到一樣的提示！」
    * 分享＆轉發：「點擊紙飛機轉發到限時動態，或是分享給閨蜜一起玩。」
    * 追蹤：「別忘了追蹤我們，每週解鎖新測驗。」
* **【主題標籤】**：放置精準關鍵字。

### 2. 動態黑底金句（巴納姆效應型）
* **【Hook】視覺降噪＆精準扎心**：畫面極簡（黑底白字），首句放上極具共鳴、似是而非的情緒金句。
* **【正文】情緒安撫**：副文分段淡入，用溫柔、理解的語氣承接扎心感。
* **【CTA】低調行動**：
    * 讚好＆留言：「如果這段話接住了你，請點個讚，或在留言區留下一顆『🖤』。」
    * 分享＆轉發：「將這份溫柔轉發給那位現在最需要聽到這些話的朋友。」
    * 追蹤：（呼籲「追蹤獲取每日平靜」）。
* **【主題標籤】**：放置情緒與療癒關鍵字。

### 3. 排盤解析乾貨教學（資訊型）
* **【Hook】展示價值**：標題直接點出學習利益。
* **【正文】步驟拆解**：限量提供乾貨，手把手教學，圖文並茂地帶領受眾看懂一個簡單的指標。
* **【CTA】預防卡關與引導轉單**：
    * 留言＆讚好：「覺得實用請按讚！跟著步驟走有卡關嗎？在留言區發問我來解答。」
    * 分享＆轉發：「轉發給對命理有興趣的朋友一起研究。」
    * 追蹤：「追蹤我看更多排盤教學。」

### 4. 小知識（共鳴型/資訊型混合）
* **【Hook】生活痛點**：從具體、微小的日常煩惱切入。
* **【正文】給出解法**：用具體事實切入，帶入正確的九宮飛星或風水方位邏輯。
* **【CTA】促發行動與共同利益**：
    * 留言＆讚好：「現在就打開指南針測量！按個讚，並在留言區告訴我你的方位對不對。」
    * 分享＆轉發：「磁場影響很大，趕緊轉發給同住的家人或室友，一起改善環境！」
    * 追蹤：「追蹤我，學習更多實用居家小知識。」

### 5. 大眾奇門占卜（上下集輪播型）
* **上集：測驗篇**（目標：拉新、互動、留存）
    * **【Hook】吸睛主標題**：拋出大眾最關心的議題。
    * **【正文】儀式感引導＆直覺選項＆品牌權威與原理背書**：引導深呼吸、提供卡牌選項。
    * **【CTA】強制互動指令＆期待感鋪墊**：留言選項＋追蹤。
* **下集：解答篇**（目標：提供價值、建立共鳴、私訊轉單）
    * **【Hook】破題主標題＆前言與破冰**：呼喚上集的朋友回來。
    * **【正文】核心價值區**：提供 A、B、C 詳細解讀。
    * **【CTA】基礎互動與轉單/變現引導**：留言反饋＋私訊預約專屬諮詢。
